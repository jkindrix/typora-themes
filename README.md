# Typora Themes

Custom themes for [Typora](https://typora.io), the minimal Markdown editor.

## Themes

### [Folio](folio/)

A print-first theme designed for PDF generation and physical printing. Optimized for documents, reports, and papers where the final output is a printed page or PDF file.

**Key features:**

- Print-first sizing (pt/mm units, no sub-pixel ambiguity)
- 10pt body text, 7.5pt code (research-backed legibility floor)
- Orphan/widow control on all text-bearing elements
- Page-break avoidance for headings, code blocks, tables, lists, and figures
- Blockquote border continuity across page breaks (`box-decoration-break: clone`)
- Table header repetition on multi-page tables
- Syntax highlighting with a muted, print-friendly palette
- Graceful degradation for monochrome printers
- System font stacks (no external fonts required)

## Installation

1. Open Typora's theme folder: **Preferences** > **Appearance** > **Open Theme Folder**
2. Copy the theme's `.css` file into the folder
3. Restart Typora
4. Select the theme from **Preferences** > **Appearance** > **Themes**
